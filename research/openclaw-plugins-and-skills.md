# OpenClaw plugins and skills notes

## Main sources
- ClawHub for skills/packages
- Local skills in workspace or installed package tree
- Plugins installed via `openclaw plugins install <source>`

## Safe adoption flow
1. shortlist
2. source/reputation review
3. code/dependency audit
4. minimal-scope test install
5. only then enable for real use

## Useful commands
```bash
openclaw skills search <term>
openclaw skills install <name>
openclaw skills list

openclaw plugins marketplace
openclaw plugins list
openclaw plugins inspect <name>
openclaw plugins install <source>
openclaw plugins enable <name>
openclaw plugins doctor
```
